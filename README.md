Overview
---------------------------------

Oregon Trail in Stata.


Quickstart
---------------------------------

Install from Github:
```stata
net install oregontrail, from(https://raw.githubusercontent.com/mdroste/stata-oregontrail/main/) replace
```

Play:
```stata
oregontrail
```
